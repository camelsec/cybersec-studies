# SQL Injection

Esta página contém informações uteis sobre SQL Injection, um tipo de ataque que permite a um invasor interferir nas consultas que um aplicativo faz ao seu banco de dados. Esse tipo de ataque pode permitir que o invasor visualize dados que normalmente não teria permissão para acessar, como dados confidenciais ou informações pessoais.
